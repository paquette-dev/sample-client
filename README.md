## Prerequisites

- Angular v16
- Node v20
- npm v10
- Angular Material v16

## Installation

Clone the repository:

```bash
git clone https://github.com/paquette-dev/sample-client.git
```

Navigate to the project directory:

```bash
cd sample-client
```

Install dependencies:

```bash
npm install
```

## Running the project

```bash
npm run start
```
